Contribute to Libree

The easist way to contribute to LibrEE is to fork it on Github, set up a local Django server on your computer (you don't need administrator access to do this) and hack away! If you have changes to submit for inclusion to LibrEE, lodge a pull request on Github, and I'll look at adding it to LibrEE as soon as possible.

Setting up LibrEE on your machine

LibrEE is supposed to be easy to set up on your own machine. The basic method looks something like this:
  • Fork the GitHub repo.
  • Clone it to your machine.
  • Install Python 2.7 and Virtualenv if you do not already have them.
  • In a fresh virtualenv, use Pip to install Django and any other required packages needed to run Libree. You may need to install the Python headers as well in order to install some of the packages like LXML.
  • Using the virtualenv, run up the server using the management script.
  • LibrEE should now be accessible at localhost:8000, or similar.
More complete and up-to-date instructions can be found in the README file at Github.
