LEDs require a given voltage to operate correctly. As diodes, they have an exponential I-V characteristic, so after the "knee voltage", they will sink a lot more current for each small increase in forward voltage. This can damage or destroy LEDs in a fraction of a second if it is not anticipated and controlled. An inline resistor is often the simplest way to limit the current for a given voltage supply and LED characteristics. {% endblock %} {% block help %}
